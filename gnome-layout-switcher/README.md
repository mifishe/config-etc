# useful multi-layout switch based on xbindkeys

1. install xbindkeys
```
apt-get install xbindkeys
```
2. run install script
```
/bin/sh install.sh
```
3. use it. 
```
  Alt+Shift+1 -> first layout in the list, 
  Alt+Shift+2 -> second, 
  ..
```
